# Snake Game

## Overview
The Snake Game is a classic arcade game where the player controls a snake that grows in length by eating food while avoiding collisions with itself and the walls.

## How to Play
- Use the arrow keys to control the direction of the snake.
- The game ends if the snake hits the wall or itself.
- Try to eat as much food as possible to grow the snake and achieve a high score.

## Features
- Simple graphics and easy-to-learn controls.
- Infinite play area with increasing difficulty.
- High score tracking.
